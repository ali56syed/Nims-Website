What is NIMS?

Software Engineering, Team 11: NIMS is a mathematical game of strategy in which two players take turns removing objects from distinct heaps. On each turn, a player must remove at least one object, and may remove any number of objects provided they all come from the same heap. The goal of the game is to be the player to remove the last object.

Let's cover some assumptions & constraints

Assumptions

  • User is currently using Android smart phone and has basic knowledge to use phone
  • User must be able to make moves based on strategic planning and mathematical patterns
  • The system contains volatile storage through SQL
  • Network issues will be ignored
  • Sound issues will be ignored
  • Text issues will be ignored

Constraints

  • Team lacks professional level android developement skills
  • Team schedule too aggressive
  • Team is not versatile in database management and programming (SQL) skills
